# Kavhdex - Your Easy Pokédex
Kavhdex is a simple Pokédex, light and totally based on PokéAPI. Its objective is allow any user to search for a Pokémon by name or nuber and view its sprite (normal or shiny) and read its official Pokédex entry.

## Functionalities
- Search for name or number in Pokédex
- Alter between normal and shiny sprites
- Navigation between previous and next Pokémon
- Exhibition of name, number and Pokédex entry
- Interface constructed with HTML, CSS and JavaScript
- Direct consumption of PokéAPI
